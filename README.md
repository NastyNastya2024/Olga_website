# Olga Website

Сайт с публичной частью и административной панелью для управления контентом.

## Структура проекта

```
├── public/          # Публичная часть сайта (HTML страницы)
├── admin/           # Админ-панель (отдельное приложение)
├── backend/         # Backend API
├── shared/          # Общие ресурсы (стили, скрипты)
├── database/        # Скрипты базы данных
└── docker-compose.yml  # Docker конфигурация
```

## Быстрый старт

### 1. Запуск инфраструктуры

```bash
# Запустить PostgreSQL и MinIO (S3)
docker-compose up -d
```

### 2. Запуск Backend

```bash
cd backend
npm install
npm start
```

### 3. Открыть сайт

- **Публичная часть**: Откройте `public/index.html` в браузере
- **Админ-панель**: Откройте `admin/login.html` в браузере

## Функционал

### Публичная часть (public/)

- Главная страница
- Страница видео
- Страница туров
- Страница клуба
- Страница блога

### Админ-панель (admin/)

#### Раздел "Видео"
- **Обычный пользователь**: Просмотр доступных видео
- **Админ**: Просмотр всех видео + добавление новых

#### Раздел "Ученики"
- **Обычный пользователь**: 
  - Просмотр своего профиля
  - Редактирование: имя, email, телефон, фото
- **Админ**:
  - Просмотр всех пользователей
  - Просмотр тарифов и дат регистрации
  - Управление пользователями

#### Раздел "Туры" (только админ)
- Добавление туров
- Редактирование описания тура
- Загрузка фото в галерею
- Управление датами и локациями

#### Раздел "Блог" (только админ)
- Добавление статей
- Редактирование статей
- Загрузка обложек
- Управление SEO полями

## API Endpoints

### Публичные
- `GET /api/public/videos` - Список доступных видео
- `GET /api/public/tours` - Список туров
- `GET /api/public/blog` - Список статей

### Админские
- `POST /api/admin/auth/login` - Вход
- `GET /api/admin/videos` - Список видео
- `POST /api/admin/videos` - Добавить видео (только админ)
- `GET /api/admin/users` - Список пользователей (только админ)
- `GET /api/admin/users/me` - Мой профиль
- `GET /api/admin/tours` - Список туров (только админ)
- `POST /api/admin/tours` - Добавить тур (только админ)
- `GET /api/admin/blog` - Список статей (только админ)
- `POST /api/admin/blog` - Добавить статью (только админ)

## Доступ к сервисам

**PostgreSQL:**
- Host: `localhost:5432`
- Database: `olga_website`
- User: `postgres`
- Password: `postgres`

**MinIO (S3):**
- S3 API: http://localhost:9000
- MinIO Console: http://localhost:9001
- Access Key: `minioadmin`
- Secret Key: `minioadmin`
