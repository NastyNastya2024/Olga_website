# Инструкция: Куда скачать видео и как загрузить в S3

## Шаг 1: Скачать видео с Яндекс Диска

### Куда скачать:
**Любое удобное место на вашем компьютере!** Например:

- **На Mac:** `~/Downloads/yoga-videos` (папка Downloads)
- **На Windows:** `C:\Users\ВашеИмя\Downloads\yoga-videos`
- **Или любая другая папка:** например, `~/Documents/videos`

### Как скачать:

1. Откройте ссылку: https://disk.yandex.ru/d/piDW3EhXQ8n46Q
2. Нажмите кнопку **"Скачать всё"** (или выберите отдельные файлы)
3. Дождитесь завершения скачивания
4. Запомните, куда сохранились файлы (обычно это папка "Загрузки" / "Downloads")

## Шаг 2: Найти путь к скачанным файлам

### На Mac:
1. Откройте Finder
2. Найдите папку "Загрузки" (Downloads)
3. Найдите папку с видео (может называться "Йога" или похоже)
4. Правой кнопкой → "Скопировать путь" или просто запомните путь

### На Windows:
1. Откройте Проводник
2. Найдите папку "Загрузки" (Downloads)
3. Найдите папку с видео
4. В адресной строке будет путь, например: `C:\Users\ВашеИмя\Downloads\Йога`

## Шаг 3: Загрузить видео в S3

### Вариант A: Если вы скачали в папку Downloads

```bash
# Перейдите в папку backend
cd backend

# Загрузите все видео из папки Downloads
node ../scripts/upload-to-s3.js ~/Downloads/Йога

# Или если папка называется по-другому:
node ../scripts/upload-to-s3.js ~/Downloads/название_папки
```

### Вариант B: Если вы знаете точный путь

```bash
cd backend

# Замените путь на ваш реальный путь
node ../scripts/upload-to-s3.js /Users/ваше_имя/Downloads/Йога
```

### Вариант C: Если вы не знаете точный путь

1. Откройте терминал
2. Перетащите папку с видео в терминал - путь появится автоматически
3. Скопируйте путь
4. Выполните:

```bash
cd backend
node ../scripts/upload-to-s3.js ВСТАВЬТЕ_СЮДА_ПУТЬ
```

## Примеры путей:

### Mac:
```bash
node ../scripts/upload-to-s3.js /Users/olga/Downloads/Йога
node ../scripts/upload-to-s3.js ~/Downloads/yoga-videos
```

### Windows (в Git Bash или WSL):
```bash
node ../scripts/upload-to-s3.js /c/Users/olga/Downloads/Йога
```

## Что произойдет:

1. Скрипт найдет все видео файлы в указанной папке
2. Загрузит их в S3 (MinIO)
3. Покажет URL каждого файла
4. Вы сможете использовать эти URL в админке

## Проверка:

После загрузки проверьте:
- MinIO Console: http://localhost:9001
- Bucket: `olga-media` → папка `videos/`

## Если возникли проблемы:

**Не могу найти папку:**
- Проверьте папку "Загрузки" / "Downloads"
- Или используйте поиск по имени файла

**Скрипт не находит файлы:**
- Убедитесь, что путь правильный
- Попробуйте перетащить папку в терминал для получения точного пути

**Ошибка подключения к S3:**
- Убедитесь, что MinIO запущен: `docker-compose up -d minio`
