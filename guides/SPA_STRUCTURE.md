# Структура SPA админ-панели

## ✅ Текущая структура (после очистки)

```
admin/
├── index.html                    # Единственная точка входа SPA
├── scripts/
│   ├── app.js                   # Главный файл приложения
│   │   ├── Layout               # Компонент макета
│   │   ├── Sidebar              # Боковое меню
│   │   ├── Header               # Шапка
│   │   ├── LoginPage            # Страница авторизации
│   │   └── DashboardPage        # Страница дашборда
│   ├── router.js                # Клиентский роутер
│   ├── guards.js                # Защита маршрутов (отключена)
│   └── pages/                   # Модули страниц
│       ├── videos.js            # Страница видео (с загрузкой файлов)
│       ├── students.js          # Страница учеников
│       ├── tours.js             # Страница туров
│       └── blog.js              # Страница блога
├── styles/
│   ├── admin.css               # Основные стили админки
│   └── modal.css               # Стили модальных окон
└── README.md                   # Документация
```

## 🎯 Как работает SPA

### 1. Точка входа
- **Файл:** `index.html`
- **Роль:** Единственный HTML файл, загружает все скрипты
- **Структура:** Пустой `<div id="app">`, контент загружается динамически

### 2. Роутинг
- **Файл:** `scripts/router.js`
- **Маршруты:**
  - `/` или `/login` → LoginPage
  - `/dashboard` → DashboardPage
  - `/videos` → VideosPage (с загрузкой файлов)
  - `/students` → StudentsPage
  - `/tours` → ToursPage
  - `/blog` → BlogPage

### 3. Загрузка видео

**Реализовано в:** `scripts/pages/videos.js`

**Функционал:**
- ✅ Форма загрузки файла через `<input type="file">`
- ✅ Прогресс-бар с процентами
- ✅ Автоматическое заполнение URL после загрузки
- ✅ Автоматическое заполнение названия из имени файла
- ✅ Интеграция с API `/api/upload`
- ✅ Поддержка больших файлов (до 20GB)
- ✅ Поддержка форматов: MP4, WebM, MOV, AVI, MKV

**Как использовать:**
1. Откройте админку → раздел "Видео"
2. Нажмите "Добавить видео"
3. Выберите файл или перетащите его
4. Дождитесь завершения загрузки
5. URL заполнится автоматически
6. Заполните остальные поля и сохраните

## 📋 Преимущества SPA структуры

1. **Один вход** - только `index.html`
2. **Быстрая навигация** - без перезагрузки страницы
3. **Модульность** - каждая страница в отдельном файле
4. **Переиспользование** - общий Layout для всех страниц
5. **Легкая поддержка** - понятная структура
6. **Загрузка файлов** - работает через SPA без проблем

## 🔧 Технические детали

### Загрузка модулей
Страницы загружаются динамически через ES6 modules:
```javascript
const VideosPage = await import('./pages/videos.js');
const content = await VideosPage.default.render();
```

### API для загрузки файлов
Используется `XMLHttpRequest` для отслеживания прогресса:
```javascript
api.uploadFile('/upload', formData, (percent) => {
    // Обновление прогресс-бара
});
```

### Рендеринг страниц
Каждая страница имеет метод `render()` и `init()`:
```javascript
export default {
    render: async () => { /* HTML */ },
    init: async () => { /* Инициализация */ }
}
```

## 🚀 Развитие

Для добавления новой страницы:
1. Создайте файл в `scripts/pages/new-page.js`
2. Добавьте маршрут в `router.route('/new-page', ...)`
3. Добавьте пункт меню в `Sidebar.render()`
