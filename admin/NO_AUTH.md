# Авторизация отключена

## Текущее состояние

Авторизация в админ-панели **отключена**. Все разделы доступны без входа.

## Как войти в админку

1. На любой публичной странице нажмите кнопку **"Вход"** в header
2. Вы автоматически попадете в админ-панель на страницу дашборда
3. Никаких логинов и паролей не требуется

## Доступные разделы

Все разделы доступны без ограничений:

- ✅ **Дашборд** - статистика
- ✅ **Видео** - просмотр и управление видео
- ✅ **Ученики** - просмотр и управление пользователями
- ✅ **Туры** - управление турами
- ✅ **Блог** - управление статьями

## Технические детали

- Guards отключены (всегда возвращают `true`)
- Проверки `isAuthenticated()` и `isAdmin()` не выполняются
- API запросы работают без токенов авторизации
- Все функции админки доступны

## Включение авторизации обратно

Если нужно вернуть авторизацию:

1. Включите guards в `admin/scripts/guards.js`
2. Добавьте проверки в роутер
3. Восстановите форму логина в `admin/scripts/app.js`
4. Включите проверку токенов в API запросах
